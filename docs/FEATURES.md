# Fonctionnalités

## Navbar : 
* Accueil
* Chercher une tier list
* Créer une tier list

## Onglet "Accueil" 
* navbar (_chercher une tier list / créer une tier list_)
* présentation du site
* les tier list tendances
* footer

## Onglet “Chercher une tier list” (SANS/AVEC recherche)
* navbar (_accueil / créer une tier list_)
* filtre de barre de recherche de tier list
* liste des tier list
* bouton "</>" pour "AVANT recherche / APRES recherche"

## Onglet "Tier list X"
* navbar (_accueil / chercher une tier list / créer une tier list_)
* titre
* bloc de la tier list (tableau)
* images de la tier list x

## Onglet “Créer une tier list”
* navbar (_accueil / chercher une tier list_)
* input : titre de la tier list
* ajouter 1 à 5 blocs de niveau
* dans chaque bloc un petit engrenage pour modifier la couleur et son nom
* une div ou on ajouter les images que l’on veut
* bouton "</>" pour le menu de création / l'onglet "Chercher une tier list" avec celle créé
